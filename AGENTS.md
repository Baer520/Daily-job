# 项目协作说明

## 语言

- 与用户沟通默认使用中文。
- 代码、命令、路径、配置项保持原文，不做不必要翻译。

## 项目概览

- 当前项目是一个 Vite + React 待办提醒应用。
- 入口文件：
  - `index.html`
  - `src/main.jsx`
  - `src/App.jsx`
  - `src/styles.css`
- 包管理文件：
  - `package.json`
  - `package-lock.json`

## 常用命令

- 安装依赖：`npm install`
- 启动开发服务：`npm run dev`
- 构建生产包：`npm run build`
- 本地预览构建产物：`npm run preview`

## 工作约定

- 修改代码前先查看现有实现，优先沿用项目当前风格。
- 保持改动范围聚焦，不做无关重构。
- 不主动覆盖或回退用户已有改动。
- 手动编辑文件时优先使用补丁方式，避免产生无关格式变化。
- 完成改动后尽量运行相关验证命令，例如 `npm run build`。

## 前端约定

- UI 组件优先保持简洁、清晰、可扫描。
- 图标优先使用项目已有的 `lucide-react`。
- 表单、按钮、筛选、列表等交互应保证移动端和桌面端都可用。
- 中文文案应使用 UTF-8 正常中文，避免乱码。
